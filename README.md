# Final Project: Emotion Detector

This repository contains the final project for the IBM Coursera Emotion Detector application.

## Project Description
The application analyzes text using the Watson NLP Emotion Prediction service and returns scores for:
- anger
- disgust
- fear
- joy
- sadness

It also returns the dominant emotion for the input text.
